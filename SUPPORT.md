# Support

## What this repository is for

`Article-Ingest` is a focused OpenClaw skill repository for article extraction and summarization workflows.

## Where to ask for help

For help, use one of these paths:

- open a GitHub issue for reproducible bugs or feature requests
- open a discussion thread if discussions are enabled later
- contact the maintainer directly for sensitive problems

## Before opening an issue

Please include as much of the following as possible:

- input type: WeChat URL / generic URL / local file / stdin / Feishu doc workflow
- exact command or usage path
- expected result
- actual result
- whether extraction was partial or blocked

## Good issue examples

- a specific WeChat article fails to extract correctly
- a local markdown file loses important sections
- output formatting is inconsistent with the documented templates
- packaged artifact installation is unclear

## Not ideal issue examples

- vague reports without sample input or reproduction steps
- requests that are outside the scope of this repository
